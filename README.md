# Energy Consumption Forecast

Machine learning project for forecasting electricity consumption using XGBoost and time-series features.

## Features

- Time-series feature engineering (lag, rolling mean, cyclical encoding)
- XGBoost regression model
- Forecast generation pipeline
- Interactive Streamlit dashboard
- Feature importance visualization

## Project Structure

dataset/
models/
src/
dashboard/

## Run the project

Install dependencies:

pip install -r requirements.txt

Run the dashboard:

streamlit run dashboard/dashboard.py
